## 说明

本仓库为演示测试用途。工具功能：

- 输入目标指标范围（Flow、HMTA、B-time、325 Mesh、水分、游离酚等），自动匹配并排序产品；
- 支持粘贴指标文本自动解析；
- 支持 Excel / CSV 导入产品数据库；
- 响应式设计，桌面和手机均可使用。

---

## 数据说明

本演示版**不内置产品数据**。

首次使用需通过 **Data Management → Import** 导入产品数据（Excel 或 CSV 格式）。

数据保存在浏览器本地 `localStorage`，不上传至任何服务器，刷新页面后仍然保留。

不同设备、不同浏览器之间数据相互独立。
